#### Spanish
-Supóngase que en el pc existe una carpeta por cada usuario.
-Crear un programa en java que cree varios procesos en paralelo.
-Cada uno de ellos debe listar el contenido contenido de la carpeta de usuario que recibe como argumento y escribir en un fichero de salida los archivos listados.
-El final de la ejecución devuelve un fichero por cada usuario.
-Una vez finalizados todos los hijos el padre imprimirá los ficheros por pantalla secuencialmente.
#### English
-In the computer exists a folder for each user.
-Create a program with java that creates several process in parallel.
-Each one of them must list the content of their assigned user folder that receive as parameter and write them in a document file.
-The end of the execution return a file for each user.
-At the end of each child process, the parent process print the document files for the screen.
